Features

Everything you need for powerful API testing and automation

Visual Node-Based Editor

Design API test workflows using an intuitive drag-and-drop interface. Connect nodes to create complex request sequences visually.

Multiple Node Types

Endpoint, Script, Assertion, Payload, Viewer, Delay, Conditional, Group, and External nodes for complete workflow control.

Environment Management

Define and switch between multiple environments (dev, staging, production) with ease. Manage API endpoints and credentials per environment.

Variable Substitution

Use environment variables and dynamic values in URLs, headers, and payloads with simple {variable} syntax.

Message Flow Tracking

Visualize and debug data flow between nodes. See exactly how data transforms through your workflow.

Custom JavaScript Engine

Built-in JavaScript execution for advanced scripting capabilities. Transform data, create custom validations, and more.

Assertion Nodes

Validate API responses with custom assertions. Ensure your APIs behave exactly as expected.

Project Management

Save and organize multiple test scenarios in project files (.rqfl). Keep your workflows organized and version-controlled.

Dark/Light Theme

Customizable UI themes for comfortable viewing. Choose the theme that works best for you.

Installation Instructions

macOS

  1. Download RequestFlow-macOS.dmg from the releases page
  2. Open the DMG file (double-click)
  3. Drag RequestFlow.app to the Applications folder shortcut
  4. Eject the DMG
  5. Important: Before first launch, remove quarantine attributes by running this in Terminal: 
    xattr -cr /Applications/RequestFlow.app
  6. On first launch: Right-click the app in Applications and select "Open"
  7. Click "Open" in the security dialog that appears
  8. The app will now run normally

Note: Since the app is currently unsigned, macOS Gatekeeper requires these extra steps. The xattr -cr command removes quarantine attributes that cause the "damaged" error, and right-click ‚Üí Open bypasses the signature requirement.

Linux

  1. Download RequestFlow-Linux-x86_64.AppImage from the releases page
  2. Make it executable: chmod +x RequestFlow-Linux-x86_64.AppImage
  3. Run it: ./RequestFlow-Linux-x86_64.AppImage

Windows

  1. Download RequestFlow-Windows-x64.zip from the releases page
  2. Extract the ZIP file to your preferred location
  3. Run RequestFlow.exe

Build from Source

RequestFlow is open source! Clone the repository and build it yourself:

git clone https://github.com/fatehmtd/RequestFlow.git
cd RequestFlow/RequestFlowApp
qmake RequestFlow.pro
make

See the README for detailed build instructions.

About RequestFlow

RequestFlow is a professional REST API testing and workflow automation tool that brings visual programming concepts to API testing. Built with Qt for cross-platform compatibility, it provides a powerful yet intuitive interface for designing, testing, and automating complex HTTP request workflows.

Created and maintained by Fateh Benmerzoug, Ph.D, RequestFlow is now open source under the MIT License, allowing developers worldwide to contribute and improve the tool.
